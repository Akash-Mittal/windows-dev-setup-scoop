# Rainmeter Skins

This folder contains skins for the [Rainmeter](https://www.rainmeter.net/) desktop customization tool.

## Skins

*   `eu_Clock.ini`: A clock skin that displays the time in the EU timezone.
*   `in_Clock.ini`: A clock skin that displays the time in the IN timezone.
*   `us_Clock.ini`: A clock skin that displays the time in the US timezone.
*   `HalfHourReminder.ini`: A skin that displays a reminder every 30 minutes.

## Usage

1.  Make sure you have [Rainmeter](https://www.rainmeter.net/) installed.
2.  Copy these `.ini` files to your Rainmeter skins folder (usually `C:\Users\YourUsername\Documents\Rainmeter\Skins`).
3.  Open the Rainmeter application, and you will see these skins listed.
4.  You can then load and position the skins on your desktop as desired.
