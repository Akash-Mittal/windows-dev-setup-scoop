# Chrome Startup Scripts

This folder contains PowerShell scripts to open Google Chrome with different user profiles and a predefined set of tabs.

## Scripts

*   `chrome-starup-personal.ps1`: Opens Chrome with the "Personal" profile.
*   `chrome-starup-professional.ps1`: Opens Chrome with the "Professional" profile.
*   `chrome-starup-personal-nica.ps1`: Opens Chrome with the "Personal" profile for "Nica" related tasks.
*   `chrome-starup-personal-banking.ps1`: Opens Chrome with the "Personal" profile for banking tasks.
*   `chrome-starup-personal-blogging.ps1`: Opens Chrome with the "Personal" profile for blogging tasks.
