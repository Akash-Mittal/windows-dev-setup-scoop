# Chrome Startup Scripts

This folder contains batch scripts to open Google Chrome with different user profiles and a predefined set of tabs.

## Scripts

*   `chrome-starup-personal.bat`: Opens Chrome with the "Personal" profile.
*   `chrome-starup-professional.bat`: Opens Chrome with the "Professional" profile.
*   `chrome-starup-personal-nica.bat`: Opens Chrome with the "Personal" profile for "Nica" related tasks.
*   `chrome-starup-personal-banking.bat`: Opens Chrome with the "Personal" profile for banking tasks.
*   `chrome-starup-personal-blogging.bat`: Opens Chrome with the "Personal" profile for blogging tasks.
